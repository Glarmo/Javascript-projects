Using very simple html with css we can achieve a clean and effective navigation bar that stays at the top of the screen. This is done by adding to the css of the list so that it works under different rules. The major change being relative position to fixed position. The second part of this script deals with a small jump that occurs when this switch happens, by simply calculating this offset the transition becomes very smooth. The rest of this text is filler.
